# CodeBEGen FE

## Overview

Build, deploy and test backend functionalities effortlessly with AI-powered assistance, optimized for speed and efficiency.


## Getting Started

In order to run this app locally, you should have the following programs installed on your computer:

-   [nodejs](https://nodejs.org/)>=20.0.0
-   [pnpm](https://yarnpkg.com/)>=9.4.0
    -   Run `pnpm --version`, if already installed skip below steps.
    -   Run `npm install pnpm`
    if you get error `pnpm.ps1 cannot be loaded because running scripts is disabled on this system`,
    -   Run `Set-ExecutionPolicy RemoteSigned -Scope CurrentUser` and then try `npm install pnpm` again

#### Clone this repository

```
git clone https://github.com/Marlinekhavele/CodeBEGen.git
cd CodeBEGen
```

#### Install dependencies

```
pnpm install
```

#### Start the server

```
pnpm dev
```
